# [KnoSys](https://knosysio.github.io/)

开放的知识管理工具。

## 特色

- 本地优先存储，可同步到云端；
- 基于 QiiDB 的[数据规范](https://qiidb.github.io/meta/zh/guides/spec/)用文件系统进行数据管理。
